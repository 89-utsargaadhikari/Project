My experience in learning the basics of web-development.

I wanted to learn coding since I was studying in Grade-8, but I have to confess I never tried to learn. It's hardly been two months since I started learning basics of html and css. I had a great time learning html and css. The basic html was really easy. I learned the starting basics from my college and my curiosity started to increase to I started to learn from scrimba.

Was it hard? My experience in the beginning was quite exciting until I got to css padding and flexbox which gave me a headache before I grasped the concept, thanks to scrimba.com which helped me to understand the concept

But I had a great time learning css animations, which I found really interesting and exciting. And there is always so much to learn in CSS.

  • HTML and CSS are lightweight.
